# Spacing

Roadie v2 uses Tailwind CSS v4's built-in spacing scale. Use utility classes
like `p-4`, `gap-2`, `mt-8` for consistent spacing.

## Tailwind spacing scale

Tailwind's default spacing scale uses a 4px base unit:

```
0    = 0px
0.5  = 2px
1    = 4px
1.5  = 6px
2    = 8px
3    = 12px
4    = 16px
5    = 20px
6    = 24px
8    = 32px
10   = 40px
12   = 48px
16   = 64px
20   = 80px
24   = 96px
```

## Using spacing with Tailwind

### Padding and gap

```tsx
// Padding
<div className="p-4">Content</div>
<div className="pt-2 pb-4 px-6">Content</div>

// Gap (flex/grid)
<div className="flex flex-col gap-4">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

// Grid with gap
<div className="grid grid-cols-2 gap-x-8 gap-y-4">
  <div>Item 1</div>
  <div>Item 2</div>
</div>
```

### Responsive spacing

```tsx
<div className="mt-2 md:mt-4 lg:mt-8 px-4 md:px-6">
  Content
</div>
```

## Best practices

1. **Use Tailwind's scale, not magic numbers** — `p-4` not `p-[17px]`
2. **Prefer `gap` over margins** — cleaner, no collapsing issues
3. **Start mobile-first** — use responsive prefixes (`md:`, `lg:`) to scale up
4. **Consistent patterns** — small gaps (1-2) for related elements, medium (4-6) for components, large (8+) for sections
