# Textarea

A multi-line text input for longer form content.

## Import

```tsx
import { Textarea } from '@oztix/roadie-components'
```

## Examples

### Default

```tsx-live
<Textarea placeholder='Enter text' />
```

### Emphasis

```tsx-live
<div className='flex flex-col gap-2'>
  <Textarea emphasis='default' placeholder='Default emphasis' />
  <Textarea emphasis='subtle' placeholder='Subtle emphasis' />
</div>
```

### Sizes

```tsx-live
<div className='flex flex-col gap-2'>
  <Textarea size='sm' placeholder='Small' />
  <Textarea size='md' placeholder='Medium (default)' />
  <Textarea size='lg' placeholder='Large' />
</div>
```

### States

Textareas start neutral, gain an accent-coloured border and ring on focus,
and switch to danger colouring when invalid.

```tsx-live
<div className='flex flex-col gap-4'>
  <div className='flex flex-col gap-1'>
    <Text size='sm' emphasis='subtle'>Default</Text>
    <Textarea placeholder='Click to focus' />
  </div>
  <div className='flex flex-col gap-1'>
    <Text size='sm' emphasis='subtle'>Invalid</Text>
    <Textarea aria-invalid='true' defaultValue='Invalid content' />
  </div>
  <div className='flex flex-col gap-1'>
    <Text size='sm' emphasis='subtle'>Disabled</Text>
    <Textarea disabled placeholder='Disabled' />
  </div>
</div>
```
