# Code

The Code component is used to display inline code snippets and blocks
of code.

## Import

```tsx
import { Code } from '@oztix/roadie-components'
```

## Examples

### Inline code

```tsx

<Code>npm install @oztix/roadie-components</Code>
```

### Emphasis

Different emphasis levels for various contexts:

```tsx

<div className='flex flex-row flex-wrap gap-2'>
  <Code emphasis='default'>default code</Code>
  <Code emphasis='strong'>strong code</Code>
  <Code emphasis='subtle'>subtle code</Code>
  <Code emphasis='subtler'>subtler code</Code>
</div>
```

### Intents

Each intent is designed for specific use cases:

- `neutral`: Default for general code snippets
- `accent`: Highlighted code
- `brand`: Brand-specific code examples
- `info`: Informational code snippets
- `success`: Successful operations or correct code
- `warning`: Warning-related code
- `danger`: Error or problematic code

```tsx

<div className='flex flex-row flex-wrap gap-2'>
  <Code intent='neutral'>neutral</Code>
  <Code intent='accent'>accent</Code>
  <Code intent='brand'>brand</Code>
  <Code intent='info'>info</Code>
  <Code intent='success'>success</Code>
  <Code intent='warning'>warning</Code>
  <Code intent='danger'>danger</Code>
</div>
```
