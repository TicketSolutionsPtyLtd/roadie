# Highlight

The Highlight component highlights substrings of text based on a search query. It is useful for search results, filtering, and drawing attention to specific text patterns.

## Import

```tsx
import { Highlight } from '@oztix/roadie-components'
```

## Examples

### Basic usage

Highlight specific text based on a query:

```tsx-live

<div className='flex flex-col gap-2'>
  <Text>
    <Highlight
      text='The quick brown fox jumps over the lazy dog'
      query='quick'
    />
  </Text>
  <Text>
    <Highlight
      text='The quick brown fox jumps over the lazy dog'
      query='fox'
    />
  </Text>
  <Text>
    <Highlight
      text='The quick brown fox jumps over the lazy dog'
      query='lazy dog'
    />
  </Text>
</div>
```

### Multiple queries

Highlight multiple different terms:

```tsx-live

<Text>
  <Highlight
    text='React, Vue, and Svelte are popular JavaScript frameworks'
    query={['React', 'Vue', 'Svelte']}
  />
</Text>
```

### Case sensitivity

By default, matching is case-insensitive. You can make it case-sensitive:

```tsx-live

<div className='flex flex-col gap-2'>
  <div className='flex flex-col gap-1'>
    <Text size='xs' emphasis='subtle'>Case insensitive (default)</Text>
    <Text>
      <Highlight
        text='Hello World'
        query='hello'
      />
    </Text>
  </div>
  <div className='flex flex-col gap-1'>
    <Text size='xs' emphasis='subtle'>Case sensitive</Text>
    <Text>
      <Highlight
        text='Hello World'
        query='hello'
        ignoreCase={false}
      />
    </Text>
  </div>
</div>
```

### Match all occurrences

By default, all matches are highlighted. You can disable this:

```tsx-live

<div className='flex flex-col gap-2'>
  <div className='flex flex-col gap-1'>
    <Text size='xs' emphasis='subtle'>Match all (default)</Text>
    <Text>
      <Highlight
        text='foo bar foo baz foo'
        query='foo'
        matchAll={true}
      />
    </Text>
  </div>
  <div className='flex flex-col gap-1'>
    <Text size='xs' emphasis='subtle'>Match first only</Text>
    <Text>
      <Highlight
        text='foo bar foo baz foo'
        query='foo'
        matchAll={false}
      />
    </Text>
  </div>
</div>
```

### In headings

Highlight works great in headings for creating eye-catching titles:

```tsx-live

<div className='flex flex-col gap-2 text-center'>
  <Heading as='h1' size='2xl'>
    <Highlight
      text='Great events need great partners'
      query='great'
      intent='accent'
    />
  </Heading>
</div>
```

### Search results example

A practical example of highlighting search results:

```tsx-live
<div className='flex flex-col gap-4'>
  <div className='flex flex-col gap-1'>
    <Heading level={3} size='sm'>
      Search results for "JavaScript"
    </Heading>
    <Text size='xs' emphasis='subtle'>3 results found</Text>
  </div>
  <div className='flex flex-col gap-1'>
    <Text>
      <Highlight text='JavaScript is a programming language' query='JavaScript' intent='accent' />
    </Text>
    <Text>
      <Highlight text='TypeScript is a typed superset of JavaScript' query='JavaScript' intent='accent' />
    </Text>
    <Text>
      <Highlight text='React is a JavaScript library for building user interfaces' query='JavaScript' intent='accent' />
    </Text>
  </div>
</div>
```
