# Text

The Text component is a fundamental building block for typography. It
supports multiple sizes, emphasis levels, and intents to accommodate
different design contexts.

## Import

```tsx
import { Text } from '@oztix/roadie-components'
```

## Examples

### Default

```tsx-live

<Text>Hello World</Text>
```

### Sizes

```tsx-live

<div className='flex flex-col gap-1'>
  <Text size='xs'>Extra small text</Text>
  <Text size='sm'>Small text</Text>
  <Text size='base'>Base text (default)</Text>
  <Text size='lg'>Large text</Text>
  <Text size='xl'>Extra large text</Text>
</div>
```

### Emphasis

Use emphasis to adjust the visual weight of text:

```tsx-live

<div className='flex flex-col gap-2'>
  <Text emphasis='strong'>Strong text</Text>
  <Text>Default text</Text>
  <Text emphasis='subtle'>Subtle text</Text>
  <Text emphasis='subtler'>Subtler text</Text>
  <Text emphasis='inverted'>Inverted text</Text>
</div>
```

Use intents with emphasis to create colored text for different contexts:

```tsx-live

<div className='grid grid-cols-1 gap-2 md:grid-cols-2 lg:grid-cols-3'>
```

### Custom elements

Text can be rendered as any HTML element or component using the `as`
prop:

```tsx-live

<div className='flex flex-col gap-2'>
  <Text as='span'>Inline text (default)</Text>
  <Text as='p'>Paragraph text</Text>
  <Text as='label'>Label text</Text>
</div>
```
