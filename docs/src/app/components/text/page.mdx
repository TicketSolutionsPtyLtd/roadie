# Text

Body text styling using utility classes for size, emphasis, and intent.

## Usage

Use raw HTML elements (`<p>`, `<span>`, etc.) with Tailwind utility classes:

```tsx
<p>Default body text</p>
<p className='text-lg'>Large text</p>
<p className='text-subtle'>Subtle secondary text</p>
<p className='text-strong'>Bold emphasis text</p>
<p className='intent-accent'>Accent-colored text</p>
<span className='text-sm'>Inline small text</span>
```

## Examples

### Default

```tsx-live

<p>Hello World</p>
```

### Emphasis

```tsx-live

<div className='flex flex-col gap-2'>
  <p className='text-strong'>Strong text</p>

  <p className='text-subtle'>Subtle text</p>
  <p className='text-subtler'>Subtler text</p>
  <p className='text-inverted'>Inverted text</p>
</div>
```

### Custom elements

Use any HTML element directly.

```tsx-live

  <span>Inline text (default)</span>

</div>
```
