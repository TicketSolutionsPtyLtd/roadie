# Button

The Button component is a fundamental building block for user interactions. It supports multiple appearances and sizes to accommodate different design contexts.

## Import

```tsx
import { Button } from '@oztix/roadie-components'
```

## Examples

### Appearances

Different button appearances for various contexts:

```tsx
live
<View gap='200' flexDirection='row'>
  <Button appearance='solid'>
    Solid Button
  </Button>
  <Button appearance='outline'>
    Outline Button
  </Button>
  <Button appearance='ghost'>
    Ghost Button
  </Button>
</View>
```

### Sizes

```tsx
live
<View gap='200' flexDirection='row' alignItems='center'>
  <Button size='sm'>Small</Button>
  <Button size='md'>Medium</Button>
  <Button size='lg'>Large</Button>
</View>
```

### Colors

Subtle is the default color for all buttons.

Use subtle for the majority of buttons. Use accent for the primary action in a set of buttons.

Use success, warning, and danger for specific situations. e.g. danger for destructive actions.

```tsx
live
```

## Usage

```tsx
live
<Button onPress={() => alert('Button pressed')}>
  Click me
</Button>
```
