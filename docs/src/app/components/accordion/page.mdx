# Accordion

Vertically stacked collapsible content sections.

## Import

```tsx
import { Accordion } from '@oztix/roadie-components'
```

## Examples

### Default

Only one item can be open at a time in single mode (the default).

```tsx-live
<Accordion type='single'>
  <Accordion.Item value='item-1'>
    <Accordion.Trigger>What is Roadie?</Accordion.Trigger>
    <Accordion.Content>
      <p className='pb-3'>Roadie is a comprehensive design system for building consistent user interfaces.</p>
    </Accordion.Content>
  </Accordion.Item>
  <Accordion.Item value='item-2'>
    <Accordion.Trigger>How do I install it?</Accordion.Trigger>
    <Accordion.Content>
      <p className='pb-3'>Install via npm or pnpm from the @oztix/roadie-components package.</p>
    </Accordion.Content>
  </Accordion.Item>
  <Accordion.Item value='item-3'>
    <Accordion.Trigger>Is it accessible?</Accordion.Trigger>
    <Accordion.Content>
      <p className='pb-3'>Yes, all components follow WAI-ARIA guidelines for accessibility.</p>
    </Accordion.Content>
  </Accordion.Item>
</Accordion>
```

### Variants

Use `appearance='contained'` for a bordered look with dividers between items.

```tsx-live
<Accordion type='single' appearance='contained'>
  <Accordion.Item value='q1'>
    <Accordion.Trigger>First question</Accordion.Trigger>
    <Accordion.Content>
      <p className='pb-3'>Answer to the first question.</p>
    </Accordion.Content>
  </Accordion.Item>
  <Accordion.Item value='q2'>
    <Accordion.Trigger>Second question</Accordion.Trigger>
    <Accordion.Content>
      <p className='pb-3'>Answer to the second question.</p>
    </Accordion.Content>
  </Accordion.Item>
</Accordion>
```

### States

Use `type='multiple'` to allow multiple items open simultaneously.

```tsx-live
<div className='grid gap-4'>
  <div className='grid gap-1'>
    <p className='text-sm text-subtle'>Single (default)</p>
    <Accordion type='single'>
      <Accordion.Item value='a'>
        <Accordion.Trigger>Section A</Accordion.Trigger>
        <Accordion.Content>
          <p className='pb-3'>Content for section A.</p>
        </Accordion.Content>
      </Accordion.Item>
      <Accordion.Item value='b'>
        <Accordion.Trigger>Section B</Accordion.Trigger>
        <Accordion.Content>
          <p className='pb-3'>Content for section B.</p>
        </Accordion.Content>
      </Accordion.Item>
    </Accordion>
  </div>
  <div className='grid gap-1'>
    <p className='text-sm text-subtle'>Multiple</p>
    <Accordion type='multiple'>
      <Accordion.Item value='a'>
        <Accordion.Trigger>Section A</Accordion.Trigger>
        <Accordion.Content>
          <p className='pb-3'>Content for section A.</p>
        </Accordion.Content>
      </Accordion.Item>
      <Accordion.Item value='b'>
        <Accordion.Trigger>Section B</Accordion.Trigger>
        <Accordion.Content>
          <p className='pb-3'>Content for section B.</p>
        </Accordion.Content>
      </Accordion.Item>
    </Accordion>
  </div>
</div>
```

### Composition

Accordion is a compound component with `Accordion.Item`, `Accordion.Trigger`, and `Accordion.Content`.

```tsx-live
<Accordion type='single'>
  <Accordion.Item value='item-1'>
    <Accordion.Trigger>What is Roadie?</Accordion.Trigger>
    <Accordion.Content>
      <p className='pb-3'>Roadie is a comprehensive design system for building consistent user interfaces.</p>
    </Accordion.Content>
  </Accordion.Item>
</Accordion>
```
