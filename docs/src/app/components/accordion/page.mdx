# Accordion

Vertically stacked collapsible content sections.

## Import

```tsx
import { Accordion } from '@oztix/roadie-components'
```

## Examples

### Default

Only one item can be open at a time in single mode (the default).

```tsx-live
<Accordion type='single'>
  <Accordion.Item value='item-1'>
    <Accordion.Trigger>What is Roadie?</Accordion.Trigger>
    <Accordion.Content>
      <p>Roadie is a comprehensive design system for building consistent user interfaces.</p>
    </Accordion.Content>
  </Accordion.Item>
  <Accordion.Item value='item-2'>
    <Accordion.Trigger>How do I install it?</Accordion.Trigger>
    <Accordion.Content>
      <p>Install via npm or pnpm from the @oztix/roadie-components package.</p>
    </Accordion.Content>
  </Accordion.Item>
  <Accordion.Item value='item-3'>
    <Accordion.Trigger>Is it accessible?</Accordion.Trigger>
    <Accordion.Content>
      <p>Yes, all components follow WAI-ARIA guidelines for accessibility.</p>
    </Accordion.Content>
  </Accordion.Item>
</Accordion>
```

### Intents

Set the color context with the `intent` prop.

```tsx-live
<div className='grid gap-6'>
  <Accordion type='single' intent='accent'>
    <Accordion.Item value='q1'>
      <Accordion.Trigger>Accent intent</Accordion.Trigger>
      <Accordion.Content>
        <p>This accordion uses the accent color palette.</p>
      </Accordion.Content>
    </Accordion.Item>
    <Accordion.Item value='q2'>
      <Accordion.Trigger>Second item</Accordion.Trigger>
      <Accordion.Content>
        <p>Intent cascades to all children.</p>
      </Accordion.Content>
    </Accordion.Item>
  </Accordion>
  <Accordion type='single' intent='danger' emphasis='subtle'>
    <Accordion.Item value='q1'>
      <Accordion.Trigger>Danger intent with subtle emphasis</Accordion.Trigger>
      <Accordion.Content>
        <p>Intents work with all emphasis levels.</p>
      </Accordion.Content>
    </Accordion.Item>
    <Accordion.Item value='q2'>
      <Accordion.Trigger>Second item</Accordion.Trigger>
      <Accordion.Content>
        <p>Intent cascades to all children.</p>
      </Accordion.Content>
    </Accordion.Item>
  </Accordion>
</div>
```

### States

Use `type='multiple'` to allow multiple items open simultaneously.

```tsx-live
<div className='grid gap-4'>
  <div className='grid gap-1'>
    <p className='text-sm text-subtle'>Single (default)</p>
    <Accordion type='single'>
      <Accordion.Item value='a'>
        <Accordion.Trigger>Section A</Accordion.Trigger>
        <Accordion.Content>
          <p>Content for section A.</p>
        </Accordion.Content>
      </Accordion.Item>
      <Accordion.Item value='b'>
        <Accordion.Trigger>Section B</Accordion.Trigger>
        <Accordion.Content>
          <p>Content for section B.</p>
        </Accordion.Content>
      </Accordion.Item>
    </Accordion>
  </div>
  <div className='grid gap-1'>
    <p className='text-sm text-subtle'>Multiple</p>
    <Accordion type='multiple'>
      <Accordion.Item value='a'>
        <Accordion.Trigger>Section A</Accordion.Trigger>

          <p>Content for section A.</p>
        </Accordion.Content>
      </Accordion.Item>
      <Accordion.Item value='b'>
        <Accordion.Trigger>Section B</Accordion.Trigger>

    </Accordion>

</div>
```

### Composition

Accordion is a compound component with `Accordion.Item`, `Accordion.Trigger`, and `Accordion.Content`.

```tsx-live
<Accordion type='single'>
  <Accordion.Item value='item-1'>
    <Accordion.Trigger>What is Roadie?</Accordion.Trigger>
    <Accordion.Content>
      <p>Roadie is a comprehensive design system for building consistent user interfaces.</p>
    </Accordion.Content>
  </Accordion.Item>
</Accordion>
```
