# Card

A container for grouping related content with optional header and footer.

## Import

```tsx
import { Card } from '@oztix/roadie-components'
```

## Examples

### Default

```tsx-live
<Card>
  <Card.Content>
    <p>Card content</p>
  </Card.Content>
</Card>
```

### Emphasis

```tsx-live
<div className='grid gap-4'>
  <Card emphasis='raised'>
    <Card.Content><p>Raised</p></Card.Content>
  </Card>
  <Card emphasis='default'>
    <Card.Content><p>Default</p></Card.Content>
  </Card>
  <Card emphasis='subtle'>
    <Card.Content><p>Subtle</p></Card.Content>
  </Card>
</div>
```

### Intents

```tsx-live
<div className='grid gap-4'>
  <Card intent='accent'>
    <Card.Content><p>Accent</p></Card.Content>
  </Card>
  <Card intent='success'>
    <Card.Content><p>Success</p></Card.Content>
  </Card>
  <Card intent='danger'>
    <Card.Content><p>Danger</p></Card.Content>
  </Card>
</div>
```

### Composition

Use `Card.Header`, `Card.Content`, and `Card.Footer` to structure content.

```tsx-live
<Card>
  <Card.Header>
    <h2 className='text-display-ui-6 text-strong'>Card title</h2>
    <p className='text-sm'>A short description of the card content.</p>
  </Card.Header>
  <Card.Content>
    <p>This is the main content area of the card.</p>
  </Card.Content>
  <Card.Footer>
    <Button size='sm'>Action</Button>
  </Card.Footer>
</Card>
```
