# Card

A container for grouping related content with optional header and footer.

## Import

```tsx
import { Card } from '@oztix/roadie-components'
```

## Examples

### Default

```tsx-live
<Card>
  <Card.Content>
    <Text>Card content</Text>
  </Card.Content>
</Card>
```

### Emphasis

```tsx-live
<div className='flex flex-col gap-4'>
  <Card emphasis='raised'>
    <Card.Content><Text>Raised</Text></Card.Content>
  </Card>
  <Card emphasis='default'>
    <Card.Content><Text>Default</Text></Card.Content>
  </Card>
  <Card emphasis='subtle'>
    <Card.Content><Text>Subtle</Text></Card.Content>
  </Card>
</div>
```

### Intents

```tsx-live
<div className='flex flex-col gap-4'>
  <Card intent='accent'>
    <Card.Content><Text>Accent</Text></Card.Content>
  </Card>
  <Card intent='success'>
    <Card.Content><Text>Success</Text></Card.Content>
  </Card>
  <Card intent='danger'>
    <Card.Content><Text>Danger</Text></Card.Content>
  </Card>
</div>
```

### Composition

Use `Card.Header`, `Card.Content`, and `Card.Footer` to structure content.

```tsx-live
<Card>
  <Card.Header>
    <Heading className='text-display-ui-6'>Card title</Heading>
    <Text size='sm'>A short description of the card content.</Text>
  </Card.Header>
  <Card.Content>
    <Text>This is the main content area of the card.</Text>
  </Card.Content>
  <Card.Footer>
    <Button size='sm'>Action</Button>
  </Card.Footer>
</Card>
```
