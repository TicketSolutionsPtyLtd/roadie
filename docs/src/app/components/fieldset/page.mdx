# Fieldset

A compound wrapper for grouping related form fields with a legend.

## Import

```tsx
import { Fieldset } from '@oztix/roadie-components'
```

## Examples

### Default

```tsx-live
<Fieldset>
  <Fieldset.Legend>Contact information</Fieldset.Legend>
  <div className='flex flex-col gap-3 mt-3'>
    <Field>
      <Field.Label htmlFor='fs-name'>Name</Field.Label>
      <Field.Input id='fs-name' placeholder='Your name' />
    </Field>
    <Field>
      <Field.Label htmlFor='fs-email'>Email</Field.Label>
      <Field.Input id='fs-email' type='email' placeholder='you@example.com' />
    </Field>
  </div>
</Fieldset>
```

### Composition

Fieldset exposes sub-components for legend, helper text, and error text.

#### With helper text

```tsx-live
<Fieldset>
  <Fieldset.Legend>Contact information</Fieldset.Legend>
  <Fieldset.HelperText>Provide your preferred contact details.</Fieldset.HelperText>
  <div className='flex flex-col gap-3 mt-3'>
    <Field>
      <Field.Label htmlFor='fs2-name'>Name</Field.Label>
      <Field.Input id='fs2-name' placeholder='Your name' />
    </Field>
    <Field>
      <Field.Label htmlFor='fs2-email'>Email</Field.Label>
      <Field.Input id='fs2-email' type='email' placeholder='you@example.com' />
    </Field>
  </div>
</Fieldset>
```

#### With error text

```tsx-live
<Fieldset>
  <Fieldset.Legend>Preferences</Fieldset.Legend>
  <Fieldset.ErrorText>Please select at least one option.</Fieldset.ErrorText>
</Fieldset>
```
