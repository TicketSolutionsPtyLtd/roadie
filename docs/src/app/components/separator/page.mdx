# Separator

The Separator component renders a visual divider between content sections. It
supports horizontal and vertical orientations and includes the appropriate
ARIA role.

## Import

```tsx
import { Separator } from '@oztix/roadie-components'
```

## Examples

### Horizontal (default)

```tsx-live
<div className='flex flex-col gap-3'>
  <Text>Content above</Text>
  <Separator />
  <Text>Content below</Text>
</div>
```

### Vertical

```tsx-live
<div className='flex flex-row items-center gap-3 h-6'>
  <Text>Left</Text>
  <Separator orientation='vertical' />
  <Text>Right</Text>
</div>
```
