# Field

A compound wrapper for labelled form fields with helper and error text.

## Import

```tsx
import { Field } from '@oztix/roadie-components'
```

## Examples

### Default

```tsx-live
<Field>
  <Field.Label htmlFor='name'>Name</Field.Label>
  <Field.Input id='name' placeholder='Enter your name' />
</Field>
```

### States

Fields inherit input interaction states. Focus shows accent colouring,
invalid shows danger colouring with error text.

```tsx-live
<div className='flex flex-col gap-6'>
  <div className='flex flex-col gap-1'>
    <p className='text-sm text-subtle'>Default</p>
    <Field>
      <Field.Label htmlFor='default-field'>Label</Field.Label>
      <Field.Input id='default-field' placeholder='Click to focus' />
      <Field.HelperText>Helper text appears below the input.</Field.HelperText>
    </Field>
  </div>
  <div className='flex flex-col gap-1'>
    <p className='text-sm text-subtle'>Invalid</p>
    <Field>
      <Field.Label htmlFor='error-field'>Label</Field.Label>
      <Field.Input id='error-field' aria-invalid='true' defaultValue='bad-value' />
      <Field.ErrorText>This field is required.</Field.ErrorText>
    </Field>
  </div>
  <div className='flex flex-col gap-1'>
    <p className='text-sm text-subtle'>Disabled</p>
    <Field>
      <Field.Label htmlFor='disabled-field'>Label</Field.Label>
      <Field.Input id='disabled-field' disabled placeholder='Cannot edit' />
    </Field>
  </div>
</div>
```

### Composition

Field exposes sub-components for label, input, textarea, helper text, and error text.

#### With helper text

```tsx-live
<Field>
  <Field.Label htmlFor='email'>Email</Field.Label>
  <Field.Input id='email' type='email' placeholder='you@example.com' />
  <Field.HelperText>We will never share your email.</Field.HelperText>
</Field>
```

#### With error text

Set `aria-invalid='true'` on the input to activate the danger state. The
input border and focus ring automatically switch to danger colours.

```tsx-live
<Field>
  <Field.Label htmlFor='password'>Password</Field.Label>
  <Field.Input id='password' type='password' aria-invalid='true' />
  <Field.ErrorText>Password must be at least 8 characters.</Field.ErrorText>
</Field>
```

#### With textarea

```tsx-live
<Field>
  <Field.Label htmlFor='bio'>Bio</Field.Label>
  <Field.Textarea id='bio' placeholder='Tell us about yourself' />
  <Field.HelperText>Maximum 200 characters.</Field.HelperText>
</Field>
```
