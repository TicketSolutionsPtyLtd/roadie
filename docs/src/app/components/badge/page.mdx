# Badge

The Badge component is a compact label used to display status, counts, or
categorical information. It supports multiple intents, emphasis levels, and
sizes.

## Import

```tsx
import { Badge } from '@oztix/roadie-components'
```

## Examples

### Emphasis

```tsx-live
<div className='flex flex-row flex-wrap gap-2'>
  <Badge emphasis='strong'>Strong</Badge>
  <Badge emphasis='subtle'>Subtle</Badge>
  <Badge emphasis='subtler'>Subtler</Badge>
</div>
```

### Intents

```tsx-live
<div className='flex flex-row flex-wrap gap-2'>
  <Badge intent='neutral'>Neutral</Badge>
  <Badge intent='accent'>Accent</Badge>
  <Badge intent='brand'>Brand</Badge>
  <Badge intent='info'>Info</Badge>
  <Badge intent='success'>Success</Badge>
  <Badge intent='warning'>Warning</Badge>
  <Badge intent='danger'>Danger</Badge>
</div>
```

### Sizes

```tsx-live
<div className='flex flex-row flex-wrap items-center gap-2'>
  <Badge size='sm'>Small</Badge>
  <Badge size='md'>Medium</Badge>
</div>
```
