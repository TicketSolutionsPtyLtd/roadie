# Mark

The Mark component is used to highlight or emphasize specific text content within your application. It renders the HTML `<mark>` element with consistent styling.

## Import

```tsx
import { Mark } from '@oztix/roadie-components'
```

## Examples

### Default

The default intent is `info`.

```tsx

<Text>
  The Mark component is used to <Mark>highlight text</Mark> within content.
</Text>
```

### Intents

Each intent is designed for specific use cases:

- `neutral`: General highlights
- `accent`: Accent highlights
- `brand`: Brand-specific highlights
- `info`: Informational highlights (default)
- `success`: Successful or positive highlights
- `warning`: Warning-related highlights
- `danger`: Error or critical highlights

```tsx

<div className='flex flex-col gap-2'>
  <Text>
    This is <Mark intent='neutral'>neutral</Mark> highlighted text.
  </Text>
  <Text>
    This is <Mark intent='accent'>accent</Mark> highlighted text.
  </Text>
  <Text>
    This is <Mark intent='brand'>brand</Mark> highlighted text.
  </Text>
  <Text>
    This is <Mark intent='info'>info</Mark> highlighted text.
  </Text>
  <Text>
    This is <Mark intent='success'>success</Mark> highlighted text.
  </Text>
  <Text>
    This is <Mark intent='warning'>warning</Mark> highlighted text.
  </Text>
  <Text>
    This is <Mark intent='danger'>danger</Mark> highlighted text.
  </Text>
</div>
```

### Nested content

Mark supports nested content including other components:

```tsx

<Text>
  You can mark text with <Mark>nested <strong>formatting</strong></Mark> inside.
</Text>
```

### In headings

Mark works great in headings for creating eye-catching titles:

```tsx

<div className='flex flex-col gap-2 text-center'>
  <Heading as='h1' size='2xl'>
    Great events need great <Mark intent='accent'>partners</Mark>
  </Heading>
  <Heading as='h2' size='lg' emphasis='subtle'>
    Not just <Mark intent='warning'>great</Mark> ticketing software. Why?
  </Heading>
</div>
```

### Use cases

Mark is useful for highlighting important information in various contexts:

```tsx

<div className='flex flex-col gap-4'>
  <div className='flex flex-col gap-1'>
    <Heading level={3} size='sm'>Search results</Heading>
    <Text>
      Found <Mark intent='accent'>5 matches</Mark> for your search query.
    </Text>
  </div>

  <div className='flex flex-col gap-1'>
    <Heading level={3} size='sm'>Status updates</Heading>
    <Text>
      Your order is <Mark intent='success'>confirmed</Mark> and will ship soon.
    </Text>
  </div>

  <div className='flex flex-col gap-1'>
    <Heading level={3} size='sm'>Warnings</Heading>
    <Text>
      This action is <Mark intent='warning'>irreversible</Mark>.
    </Text>
  </div>
</div>
```
