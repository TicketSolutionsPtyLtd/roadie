# IconButton

The IconButton component is a square button optimized for displaying icons. It maintains equal width and height across all sizes, making it perfect for icon-only actions.

## Import

```tsx
import { IconButton } from '@oztix/roadie-components'
```

## Examples

### Basic usage

IconButton requires an `aria-label` for accessibility since it typically does not contain visible text.

```tsx-live

<div className='flex flex-row flex-wrap gap-2'>
  <IconButton aria-label='Add item' emphasis='strong' intent='accent'>
    +
  </IconButton>
  <IconButton aria-label='Edit item' emphasis='default'>
    ✏️
  </IconButton>
  <IconButton aria-label='Delete item' emphasis='subtle' intent='danger'>
    ×
  </IconButton>
</div>
```

### Emphasis

Different button emphasis levels for various contexts:

```tsx-live

<div className='flex flex-row flex-wrap gap-2'>
  <IconButton aria-label='Strong' emphasis='strong'>
    ★
  </IconButton>
  <IconButton aria-label='Default' emphasis='default'>
    ★
  </IconButton>
  <IconButton aria-label='Subtle' emphasis='subtle'>
    ★
  </IconButton>
  <IconButton aria-label='Subtler' emphasis='subtler'>
    ★
  </IconButton>
</div>
```

### Sizes

IconButton maintains square proportions across all sizes. The default size is `icon-md`.

```tsx-live

<div className='flex flex-row flex-wrap items-center gap-2'>
  <IconButton aria-label='Extra small' size='icon-xs'>
    ✓
  </IconButton>
  <IconButton aria-label='Small' size='icon-sm'>
    ✓
  </IconButton>
  <IconButton aria-label='Medium' size='icon-md'>
    ✓
  </IconButton>
  <IconButton aria-label='Large' size='icon-lg'>
    ✓
  </IconButton>
</div>
```

### Intents

Each intent is designed for specific use cases:

- `neutral`: Default for most actions
- `accent`: Primary or highlighted actions
- `brand`: Brand-specific actions
- `info`: Informational or help-related actions
- `success`: Positive or completion actions
- `warning`: Cautionary actions
- `danger`: Destructive or high-risk actions

```tsx-live

<div className='flex flex-col gap-2'>
  <div className='flex flex-row flex-wrap gap-2'>
    <IconButton aria-label='Neutral' emphasis='strong' intent='neutral'>✓</IconButton>
    <IconButton aria-label='Accent' emphasis='strong' intent='accent'>✓</IconButton>
    <IconButton aria-label='Brand' emphasis='strong' intent='brand'>✓</IconButton>
    <IconButton aria-label='Info' emphasis='strong' intent='info'>✓</IconButton>
    <IconButton aria-label='Success' emphasis='strong' intent='success'>✓</IconButton>
    <IconButton aria-label='Warning' emphasis='strong' intent='warning'>✓</IconButton>
    <IconButton aria-label='Danger' emphasis='strong' intent='danger'>✓</IconButton>
  </div>
  <div className='flex flex-row flex-wrap gap-2'>
    <IconButton aria-label='Neutral' emphasis='default' intent='neutral'>✓</IconButton>
    <IconButton aria-label='Accent' emphasis='default' intent='accent'>✓</IconButton>
    <IconButton aria-label='Brand' emphasis='default' intent='brand'>✓</IconButton>
    <IconButton aria-label='Info' emphasis='default' intent='info'>✓</IconButton>
    <IconButton aria-label='Success' emphasis='default' intent='success'>✓</IconButton>
    <IconButton aria-label='Warning' emphasis='default' intent='warning'>✓</IconButton>

</div>
```

## Usage

```tsx-live

<IconButton
  aria-label='Add to favorites'
  emphasis='strong'
  intent='accent'
  onClick={() => alert('Added to favorites')}
>
  ♥
</IconButton>
```

## Accessibility

Always provide an `aria-label` prop to describe the button's action, since icon buttons typically do not contain visible text labels.

```tsx-live
<div className='flex flex-row flex-wrap gap-4 items-center'>
  <div className='flex flex-col gap-1 items-center'>
    <IconButton aria-label='Close dialog'>×</IconButton>
    <Text size='xs' emphasis='subtle'>With aria-label</Text>
  </div>
  <div className='flex flex-col gap-1 items-center'>
    <IconButton aria-label=''>×</IconButton>
    <Text size='xs' intent='danger' emphasis='subtle'>Missing label</Text>
  </div>
</div>
```
