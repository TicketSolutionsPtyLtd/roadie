# IconButton

The IconButton component is a square button optimized for displaying icons. It maintains equal width and height across all sizes, making it perfect for icon-only actions.

## Import

```tsx
import { IconButton } from '@oztix/roadie-components'
```

## Examples

### Basic usage

IconButton requires an `aria-label` for accessibility since it typically does not contain visible text.

```tsx-live
<div className='flex flex-row flex-wrap gap-2'>
  <IconButton aria-label='Add item' emphasis='strong' intent='accent'>
    <Plus weight='bold' />
  </IconButton>
  <IconButton aria-label='Edit item' emphasis='default'>
    <PencilSimple weight='bold' />
  </IconButton>
  <IconButton aria-label='Delete item' emphasis='subtle' intent='danger'>
    <Trash weight='bold' />
  </IconButton>
</div>
```

```tsx-live
  <Heart weight='bold' />
```


## Accessibility

Always provide an `aria-label` prop to describe the button's action, since icon buttons typically do not contain visible text labels.

```tsx-live
<div className='flex flex-row flex-wrap gap-4 items-center'>
  <div className='flex flex-col gap-1 items-center'>
    <IconButton aria-label='Close dialog'><XCircle weight='bold' /></IconButton>
    <Text size='xs' emphasis='subtle'>With aria-label</Text>
  </div>
  <div className='flex flex-col gap-1 items-center'>
    <IconButton aria-label=''><XCircle weight='bold' /></IconButton>
    <Text size='xs' intent='danger' emphasis='subtle'>Missing label</Text>
  </div>
</div>
```
