# IconButton

A square button for icon-only actions.

## Import

```tsx
import { IconButton } from '@oztix/roadie-components'
```

## Examples

### Default

Always provide an `aria-label` since there is no visible text.

```tsx-live
<IconButton aria-label='Add item'>
  <Plus weight='bold' />
</IconButton>
```

### Emphasis

```tsx-live
<div className='flex flex-row flex-wrap gap-2'>
  <IconButton aria-label='Strong' emphasis='strong'>
    <Star weight='bold' />
  </IconButton>
  <IconButton aria-label='Normal' emphasis='normal'>
    <Star weight='bold' />
  </IconButton>
  <IconButton aria-label='Subtle' emphasis='subtle'>
    <Star weight='bold' />
  </IconButton>
  <IconButton aria-label='Subtler' emphasis='subtler'>
    <Star weight='bold' />
  </IconButton>
</div>
```

### Sizes

The default size is `icon-md`. All sizes maintain square proportions.

```tsx-live
<div className='flex flex-row flex-wrap items-center gap-2'>
  <IconButton aria-label='Extra small' size='icon-xs'>
    <Heart weight='bold' />
  </IconButton>
  <IconButton aria-label='Small' size='icon-sm'>
    <Heart weight='bold' />
  </IconButton>
  <IconButton aria-label='Medium' size='icon-md'>
    <Heart weight='bold' />
  </IconButton>
  <IconButton aria-label='Large' size='icon-lg'>
    <Heart weight='bold' />
  </IconButton>
</div>
```

## Guidelines

- Always provide an `aria-label` to describe the action since there is no visible text.
- For icon links that navigate, use [LinkIconButton](/components/link-icon-button) instead.
