# Select

The Select component provides an accessible dropdown for choosing from a list
of options. It is built on Base UI primitives and supports multiple sizes,
emphasis levels, and intents.

## Import

```tsx
import { Select } from '@oztix/roadie-components'
```

## Examples

### Basic select

```tsx-live
<Select defaultValue='apple'>
  <Select.Trigger />
  <Select.Popup>
    <Select.Item value='apple'>Apple</Select.Item>
    <Select.Item value='banana'>Banana</Select.Item>
    <Select.Item value='cherry'>Cherry</Select.Item>
  </Select.Popup>
</Select>
```
