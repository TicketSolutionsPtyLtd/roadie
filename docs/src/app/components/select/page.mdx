# Select

A dropdown for choosing from a list of options.

## Import

```tsx
import { Select } from '@oztix/roadie-components'
```

## Examples

### Default

```tsx-live
<Select defaultValue='apple'>
  <Select.Trigger />
  <Select.Popup>
    <Select.Item value='apple'>Apple</Select.Item>
    <Select.Item value='banana'>Banana</Select.Item>
    <Select.Item value='cherry'>Cherry</Select.Item>
  </Select.Popup>
</Select>
```

### States

The select trigger uses a raised surface with the same interaction states
as other form controls. Focus shows accent colouring, disabled reduces
opacity.

```tsx-live
<div className='grid gap-4'>
  <div className='grid gap-1'>
    <p className='text-sm text-subtle'>Default</p>
    <Select defaultValue='option'>
      <Select.Trigger />
      <Select.Popup>
        <Select.Item value='option'>Select an option</Select.Item>
      </Select.Popup>
    </Select>
  </div>
  <div className='grid gap-1'>
    <p className='text-sm text-subtle'>Disabled</p>
    <Select defaultValue='disabled'>
      <Select.Trigger disabled />
      <Select.Popup>
        <Select.Item value='disabled'>Cannot change</Select.Item>
      </Select.Popup>
    </Select>
  </div>
</div>
```

### Composition

Select exposes sub-components for trigger, popup, item, and label.

```tsx-live
<div className='grid gap-1.5'>
  <Select.Label>Favourite fruit</Select.Label>
  <Select defaultValue='apple'>
    <Select.Trigger />
    <Select.Popup>
      <Select.Item value='apple'>Apple</Select.Item>
      <Select.Item value='banana'>Banana</Select.Item>
      <Select.Item value='cherry'>Cherry</Select.Item>
    </Select.Popup>
  </Select>
</div>
```
