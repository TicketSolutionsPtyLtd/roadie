# Heading

Semantic heading element with display text styles, emphasis, and intents.

## Import

```tsx
import { Heading } from '@oztix/roadie-components'
```

## Examples

### Default

```tsx-live

<Heading>Default heading</Heading>
```

### Emphasis

```tsx-live

<div className='flex flex-col gap-2'>
  <Heading emphasis='strong'>Strong emphasis heading</Heading>
  <Heading emphasis='default'>Default emphasis heading</Heading>
  <Heading emphasis='subtle'>Subtle emphasis heading</Heading>
  <Heading emphasis='subtler'>Subtler emphasis heading</Heading>
  <Heading emphasis='inverted'>Inverted emphasis heading</Heading>
</div>
```

### Intents

### With font weight

Use Tailwind font-weight utilities to adjust weight independently from the text style.

```tsx-live

<div className='flex flex-col gap-2'>
  <Heading className='font-black'>Strong weight heading</Heading>
  <Heading className='font-bold'>Bold weight heading</Heading>
  <Heading className='font-medium'>Medium weight heading</Heading>
  <Heading className='font-normal'>Normal weight heading</Heading>
</div>
```

### With custom element

Combine `as` with a `text-display-*` class to decouple semantics from visual presentation.

```tsx-live

<Heading as='h1' className='text-display-ui-3'>Page title rendered as h1</Heading>
```
