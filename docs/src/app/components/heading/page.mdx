# Heading

A heading component for titles and section headers. It supports
semantic heading levels, multiple sizes, emphasis levels, and intents.

## Import

```tsx
import { Heading } from '@oztix/roadie-components'
```

## Examples

### Heading levels

Use the `level` prop to set the semantic heading level (renders the corresponding h1-h6 element):

```tsx-live

<div className='flex flex-col gap-2'>
  <Heading level={1}>Heading level 1</Heading>
  <Heading level={2}>Heading level 2</Heading>
  <Heading level={3}>Heading level 3</Heading>
  <Heading level={4}>Heading level 4</Heading>
  <Heading level={5}>Heading level 5</Heading>
  <Heading level={6}>Heading level 6</Heading>
</div>
```

### Sizes

Use the `size` prop to control the visual size independently from the semantic level:

```tsx-live

<div className='flex flex-col gap-2'>
  <Heading size='xs'>Extra small heading</Heading>
  <Heading size='sm'>Small heading</Heading>
  <Heading size='base'>Base heading</Heading>
  <Heading size='lg'>Large heading</Heading>
  <Heading size='xl'>Extra large heading</Heading>
  <Heading size='2xl'>2XL heading</Heading>
  <Heading size='3xl'>3XL heading</Heading>
  <Heading size='4xl'>4XL heading</Heading>
</div>
```

### Intents

Each intent is designed for specific use cases:

- `neutral`: Default for general content
- `accent`: Highlighted sections
- `brand`: Brand-specific headings
- `info`: Informational headings
- `success`: Positive or completion headings
- `warning`: Cautionary headings
- `danger`: Error or critical headings

```tsx-live

<div className='flex flex-col gap-2'>
  <Heading intent='neutral'>Neutral heading</Heading>
  <Heading intent='accent'>Accent heading</Heading>
  <Heading intent='brand'>Brand heading</Heading>
  <Heading intent='info'>Info heading</Heading>
  <Heading intent='success'>Success heading</Heading>
  <Heading intent='warning'>Warning heading</Heading>
  <Heading intent='danger'>Danger heading</Heading>
</div>
```

### Emphasis

Use the `emphasis` prop to adjust the visual weight of a heading:

```tsx-live

<div className='flex flex-col gap-2'>
  <Heading emphasis='strong'>Strong emphasis heading</Heading>
  <Heading emphasis='default'>Default emphasis heading</Heading>
  <Heading emphasis='subtle'>Subtle emphasis heading</Heading>
  <Heading emphasis='subtler'>Subtler emphasis heading</Heading>
  <Heading emphasis='inverted'>Inverted emphasis heading</Heading>
</div>
```

### Custom element

Use the `as` prop to override the rendered HTML element:

```tsx-live

<Heading as='h1' size='2xl'>Page title rendered as h1</Heading>
```
