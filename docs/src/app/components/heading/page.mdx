# Heading

Semantic heading elements with display text styles, emphasis, and intents. Use raw `<h1>`-`<h6>` elements with Tailwind utility classes.

## Usage

```tsx
<h1 className='text-display-prose-1 text-strong'>Page title</h1>
<h2 className='text-display-ui-3 text-strong'>Section heading</h2>
<h3 className='text-display-ui-4 text-strong'>Subsection</h3>
```

## Examples


### With font weight

Use Tailwind font-weight utilities to adjust weight independently from the text style.

```tsx-live

<div className='flex flex-col gap-2'>
  <h2 className='text-display-ui text-strong font-black'>Strong weight heading</h2>
  <h2 className='text-display-ui text-strong font-bold'>Bold weight heading</h2>
  <h2 className='text-display-ui text-strong font-medium'>Medium weight heading</h2>
  <h2 className='text-display-ui text-strong font-normal'>Normal weight heading</h2>
</div>
```

### With custom element

Combine any heading element with a `text-display-*` class to decouple semantics from visual presentation.

```tsx-live

<h1 className='text-display-ui-3 text-strong'>Page title rendered as h1</h1>
```
